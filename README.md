# 🎓 AI Student Performance Prediction Dashboard

An **AI-powered Data Science project** that predicts a student's academic performance using Machine Learning and presents insights through an interactive dashboard.

This project combines **Machine Learning, Data Visualization, and Web App Development** to analyze factors affecting student performance and predict scores in real time.

---

## 🚀 Project Overview

The **Student Performance Prediction System** uses a trained Machine Learning model to estimate a student's overall score based on key academic and behavioral factors such as:

* Study Hours
* Attendance Percentage
* Subject Scores
* Learning Patterns

The results are displayed in a **beautiful interactive dashboard built with Streamlit**.

## 📊 Dashboard Analytics

The dashboard provides several insights including:

* **Score Distribution Visualization**
* **Study Hours vs Performance Analysis**
* **Feature Correlation Heatmap**
* **Actual vs Predicted Score Comparison**
* **Interactive Charts**

These visualizations help understand **what factors influence student success the most**.

---

## 🧠 Machine Learning Model

The prediction system is built using:

* Linear Regression Model
* Data preprocessing and feature engineering
* Model evaluation using prediction analysis

The trained model is stored and loaded using **joblib** for fast predictions inside the web application.

## ▶ How to Run the Project

1. Clone the repository
2. Install dependencies

```
pip install -r requirements.txt
```

3. Run the Streamlit app

```
streamlit run app.py
```

The dashboard will open in your browser.
